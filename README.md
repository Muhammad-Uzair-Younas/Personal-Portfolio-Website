# Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML5, CSS3, Bootstrap 5, and JavaScript** to showcase my projects, technical skills, education, certifications, and professional experience.

## Live Demo 

 **Portfolio:**  
https://muhammad-uzair-younas.github.io/Personal-Portfolio-Website/

## Features

- Responsive design for desktop, tablet, and mobile
- Modern UI with Bootstrap 5
- Smooth scrolling navigation
- Hero section with profile image
- About Me section
- Skills section
- Experience section
- Project showcase with image carousel
- Project demo videos inside Bootstrap modals
- Education section
- Certifications section
- Contact section
- Scroll animations using AOS
- GitHub Pages deployment

## Installation

Clone the repository

```bash
git clone https://github.com/Muhammad-Uzair-Younas/Personal-Portfolio-Website.git
```

Open the project

```bash
cd Personal-Portfolio-Website
```

Run by opening

```
index.html
```

or use the VS Code Live Server extension.
